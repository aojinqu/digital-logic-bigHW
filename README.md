# digital-logic-bigHW
数字逻辑大作业——基于 FPGA 、蓝牙、MP3、VGA模块的下落式音游

耳机中播放音乐，玩家通过手机蓝牙控制传到板子，会根据判定给分，每10分在管子上显示。
